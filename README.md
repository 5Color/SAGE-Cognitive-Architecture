## 🇬🇧 English

SAGE is an **experimental cognitive architecture prototype** exploring modular organs, adaptive routing, memory, reflection, and teacher-student language organ training.

### Core Goals
- Achieve strong adaptability with **less computation**
- Allow organs to naturally specialize
- Biological inspiration: Memory, Routing, Reflection, Energy Homeostasis, etc.

This is an early-stage prototype. Long-term goal is to move towards AGI direction.

---

## 📁 Project Structure

- `sage_core/` — Core architecture (organs, routing, memory, etc.)
- `sage_runtime/` — Runtime execution
- `sage_ui/` — User interface
- `configs/` — Configuration files
- `experiments/` — Experiment logs
- `memory/` — Memory modules
- `tools/` — Utilities
